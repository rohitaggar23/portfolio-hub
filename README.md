# Rohit Aggarwal Portfolio

Static GitHub Pages portfolio for five AI/ML systems projects.

The site is served from `docs/` and highlights:

- Real dataset outputs for Orion, Atlas, and Sentinel.
- Existing generated outputs for Rescue Swarm AI and Restaurant Hybrid Recommender.
- Project metrics, figures, artifact links, and repository links.
- Editable profile, GitHub, LinkedIn, email, and resume fields in `docs/data/config.json`.

## Deploy

1. Open repository settings on GitHub.
2. Go to Pages.
3. Select `main` as the branch.
4. Select `/docs` as the publishing folder.

## Configure

Edit `docs/data/config.json` and replace:

- `<your-username>`
- `<your-profile>`
- `<your-email>`
- `resumeUrl` when a hosted resume file is available
