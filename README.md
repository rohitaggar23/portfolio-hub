# Rohit Aggarwal Portfolio

Static GitHub Pages portfolio for Rohit Aggarwal's AI/ML systems work.

The site is served from `docs/` and includes:

- All repositories under `repos/`.
- Real dataset outputs for Orion, Atlas, and Sentinel.
- Existing generated outputs for Rescue Swarm AI and Restaurant Hybrid Recommender.
- Profile photograph and downloadable resume.
- Publications, education, experience, project metrics, generated figures, and artifact links.
- Animated recruiter-facing landing page with project filters and detail modals.

## Deploy

1. Open repository settings on GitHub.
2. Go to Pages.
3. Select `main` as the branch.
4. Select `/docs` as the publishing folder.

## Configure

Edit `docs/data/config.json` to update profile links, resume path, education, publications, or repository names.
